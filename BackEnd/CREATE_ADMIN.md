# Create Admin User

Run this command in your terminal:

```powershell
cd C:\Users\HP\Downloads\Nuit\BackEnd
venv\Scripts\activate
python manage.py createsuperuser
```

When prompted, enter:
- **Phone (username):** `+22242038210` (or any phone number)
- **Full name:** Your name (e.g., `Admin User`)
- **Email:** Press Enter to skip
- **Password:** Enter a secure password (you'll be asked twice)

**Note:** The phone number will be your login username for the admin panel.
